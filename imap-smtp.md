# SMTP and IMAP Server and Ports

## Gmail
##### SMTP Server `smtp.gmail.com`
##### SMTP Port `465` (SSL)
##### SMTP Port `587` (TLS)
##### IMAP Server `imap.gmail.com`
##### IMAP Port `993`

## Hotmail
##### Server `smtp.live.com`
##### Port `25` or `465`
##### POP3 Server `pop3.live.com`
##### POP3 Port `995`

## Zoho Mail
##### SMTP Server `zoho.gmail.com`
##### SMTP Port `465` (SSL)
##### SMTP Port `587` (TLS)
##### IMAP Server `imappro.zoho.com`
##### IMAP Port `993`

## Yahoo Mail
##### SMTP Server `smtp.mail.yahoo.com`
##### SMTP Port `465` (SSL)
##### SMTP Port `587` (TLS)
##### IMAP Server `iimap.mail.yahoo.com`
##### IMAP Port `993`
